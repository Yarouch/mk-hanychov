# 🚀 NÁVOD: Jak nahrát na GitHub Pages (krok za krokem)

## ⏱️ Čas: 10 minut | 💰 Cena: ZDARMA

---

## 📋 KROK 1: Vytvoř GitHub účet (pokud nemáš)

1. Jdi na **https://github.com**
2. Klikni **Sign up** (vpravo nahoře)
3. Zadej:
   - Email
   - Heslo
   - Username (například: `jaroslav-podsednik` nebo `mk-hanychov`)
4. Potvrď email
5. **Hotovo!** ✅

---

## 📁 KROK 2: Vytvoř nový repozitář

1. Po přihlášení klikni **"+" → New repository** (vpravo nahoře)

2. Vyplň:
   - **Repository name:** `mk-horni-hanychov` (nebo jiný název)
   - **Description:** `Digitální nástroje pro Místní komisi Horní Hanychov`
   - **Public** ✓ (musí být public pro GitHub Pages zdarma)
   - **Add a README file** ✓ (zaškrtni)

3. Klikni **Create repository**

4. **Hotovo!** ✅ Máš repozitář na adrese:
   ```
   https://github.com/[tvůj-username]/mk-horni-hanychov
   ```

---

## 📤 KROK 3: Nahraj soubory

### 3A) Přejmenuj soubory (na Tvém PC):

```
zapis_WITH_MAP_v3_FINAL.html           → zapis.html
formular_podnetu_WITH_MAP_v3_FINAL.html → podnet.html
```

### 3B) Nahraj do GitHubu:

1. V repozitáři klikni **"Add file" → "Upload files"**

2. **Přetáhni** nebo vyber tyto soubory:
   - `index.html`
   - `zapis.html`
   - `podnet.html`
   - `README.md`

3. Do pole "Commit message" napiš: `První verze - zápisy a podněty s mapami`

4. Klikni **Commit changes**

5. **Hotovo!** ✅ Soubory jsou nahrané

---

## 🌐 KROK 4: Zapni GitHub Pages

1. V repozitáři klikni na **Settings** (vpravo nahoře)

2. V menu vlevo klikni **Pages**

3. V sekci **"Source"** nastav:
   - **Branch:** `main` (vyber z dropdown)
   - **Folder:** `/ (root)` (nech)

4. Klikni **Save**

5. **Počkej 1-2 minuty** ⏳

6. Refresh stránku → mělo by se zobrazit:
   ```
   ✅ Your site is live at https://[username].github.io/mk-horni-hanychov/
   ```

7. **Hotovo!** ✅ Web je online!

---

## 🎉 KROK 5: Otevři web

1. Klikni na odkaz:
   ```
   https://[tvůj-username].github.io/mk-horni-hanychov/
   ```

2. Měl by se zobrazit:
   ```
   🏛️ Místní komise
   Horní Hanychov
   Liberec

   [📝 Zápis z jednání]
   [📋 Formulář podnětu]
   [📊 Dashboard podnětů] (zatím disabled)
   ```

3. **Funguje!** 🎊

## 🆘 Časté problémy

### ❌ "404 - File not found"
**Řešení:** Počkej 2-3 minuty po zapnutí GitHub Pages, pak refresh

### ❌ "Stránka se nezobrazuje správně"
**Řešení:** Zkontroluj že soubory mají správné názvy:
- `index.html` (ne `index.HTML`)
- `zapis.html` (ne `zapis_WITH_MAP...`)

### ❌ "Cannot enable GitHub Pages"
**Řešení:** Repozitář musí být **Public**, ne Private

---

## 🔄 Jak aktualizovat web (později)

1. Jdi do repozitáře na GitHubu
2. Klikni na soubor který chceš změnit (např. `index.html`)
3. Klikni na **✏️ ikonu** (Edit this file)
4. Udělej změny
5. Klikni **Commit changes**
6. **Počkaj 1 minutu** → změny se projeví na webu

**NEBO:**

1. Upload files → Vyber novou verzi
2. Přepiš starý soubor
3. Commit
